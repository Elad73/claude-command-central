# Python v1 (archived)

This is the original stdlib-only Python implementation of Claude Command Central.
The project has been rewritten in TypeScript + Ink (see the repo root).

This directory is preserved as a reference and a rollback option. It is not maintained.

## Running the archived version

```bash
cd legacy/python
python3 command_central.py watch --feed runtime/demo-feed.jsonl
```

See the original design notes in the root `PRD.md`.
